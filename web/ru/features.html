Функции демонстрационной версии

Регистрация нового пользователя

Пользователи могут зарегистрироваться через чат, вводя данные на своём родном языке (голосом или текстом).

  • Email.
  • Парольную фразу.
  • Язык приложения.

ChatGPT структурирует информацию и отправляет её через POST-запросы в веб-сервис для создания аккаунта. После регистрации пользователь получает уникальный PIN и парольную фразу.

Верификация email

Для завершения регистрации веб-сервис отправляет письмо с токеном для подтверждения.

В демо-версии email отправляется только на английском языке.

Пользователь подтверждает email через предоставленную ссылку, после чего его аккаунт активируется.

Редактирование профиля

Пользователи могут запросить изменение данных через чат на своём родном языке.

Веб-сервис формирует одноразовую ссылку для редактирования, которую пользователь получает по email (в демо-версии только на английском языке).

Через форму можно изменить:

  • Язык интерфейса приложения.
  • Парольную фразу.

Восстановление доступа

Если пользователь забывает PIN или парольную фразу, он может запросить восстановление данных через чат.

ChatGPT отправляет запрос веб-сервису, который формирует и отправляет ссылку для сброса данных.

Взаимодействие через email в демо-версии только на английском языке.

Отправка тестового email

Зарегистрированные пользователи могут отправить тестовое письмо через чат.

В демо-версии письмо отправляется только на английском языке.
