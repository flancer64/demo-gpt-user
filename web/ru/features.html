Функции демо-приложения

Оглавление

  • Регистрация нового пользователя
  • Верификация email
  • Редактирование данных
  • Отправка тестового email себе
  • Отправка сообщения владельцу бизнеса

Регистрация нового пользователя

Пользователи могут зарегистрироваться через чат, вводя данные на своём родном языке (голосом или текстом).

Для регистрации требуется указать:

  • Email.
  • Имя.
  • Парольную фразу.
  • Язык приложения.

ChatGPT структурирует информацию и отправляет её через POST-запросы в веб-сервис для создания аккаунта. После регистрации пользователь получает уникальный PIN и парольную фразу.

Верификация email

Для завершения регистрации веб-сервис отправляет письмо для подтверждения email-адреса. Пользователь подтверждает email через предоставленную ссылку, после чего его аккаунт активируется, и он получает возможность использовать другие сервисы через действия.

Редактирование данных

Пользователи могут запросить редактирование своих данных через чат, указав PIN-код (если помнят) или email-адрес. На указанный email отправляется одноразовая ссылка для входа в интерфейс редактирования.

С помощью этой ссылки пользователь может:

  • Изменить своё имя.
  • Изменить язык интерфейса приложения.
  • Просмотреть свой PIN-код.
  • Задать новую парольную фразу.

Этот процесс объединяет редактирование данных и восстановление доступа, обеспечивая простой и безопасный способ управления учётной записью.

Отправка тестового email себе

Зарегистрированные пользователи могут отправить тестовое письмо через чат. Эта функция помогает проверить корректность настроек аккаунта и работоспособность email-сервиса.

Отправка сообщения владельцу бизнеса

Зарегистрированные пользователи могут отправить email-сообщение владельцу бизнеса. Это позволяет задать вопросы, оставить отзывы или предложить сотрудничество.
