New User Registration

Users can register through the chat by providing their details in their native language (either by voice or text).

To register, users need to provide the following:

  • Email address.
  • Name.
  • Password phrase.
  • Preferred application language.

ChatGPT structures this information and sends it as POST requests to the web service for account creation. After registration, the user receives a unique PIN and password phrase.

Email Verification

To complete the registration, the web service sends a verification email to the provided email address. The user confirms their email using the provided link, after which their account is activated, granting them access to additional services through actions.

Access Restore and Profile Editing

Users can request to edit their account data or restore access through the chat by providing their PIN (if remembered) or their email address. A one-time link is sent to the provided email, allowing access to the editing interface.

Using this link, users can:

  • Change their name.
  • Modify the application's interface language.
  • View their PIN code.
  • Set a new password phrase.

This process combines access recovery and profile editing into a simple and secure method for managing user accounts.

Sending a Test Email

Registered users can send a test email to themselves through the chat. This feature helps verify account settings and ensure the email service is functioning correctly.

Sending a Message to the Business Owner

Registered users can send an email message to the business owner. This allows them to ask questions, leave feedback, or propose collaboration opportunities.
