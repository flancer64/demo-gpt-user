GPT Chat: Voice Authentication

About the Project

This project demonstrates the integration of a custom GPT chat with an external web application. You can see how the chat interacts with the application through actions, processing user requests, and forwarding them to the web application.

Experimental Feature

The key feature of this project is experimental voice registration and authentication. It allows users to input their authentication data (a digital PIN and a password phrase) using their voice. This demonstrates the potential of voice interfaces in cases where standard methods, such as OAuth 2.0, are not applicable.

Although registration is simplified by manually entering an email address, the system also supports letter-by-letter email input using voice. This makes the experiment unique and distinguishes it from existing solutions.

Features

The demo application provides functionality for new user registration, email verification, profile data editing, access recovery, and sending test emails. All features are available through voice interactions with the chat.
